FAQs

What kind of files does the site uses ?

The site uses seismic data files. Check the Obspy obspy.core.stream.read function for the supported formats

What tools are used to build this website ?

The technologies used to build this website consists of front-end development tools such as pure HTML, CSS, Javascript, frameworks like Bootstrap and tools related to the back-end or server side such as the Flask Python library.

What is the site about ?

The site is about earthquakes. It is organized into earthquake related articles and interactive tools. Dive into the articles to gain deeper insights into various seismological concepts, and feel free to utilize the available interactive tools that let you explore and apply different seismological computations.

Will the site be regularly updated with new content?

Yes, i am committed to regularly update the site with fresh articles, tools, and the latest insights in the field of seismology. Be sure to check back often to discover new additions that will continue to enrich your understanding and exploration of earthquakes.
